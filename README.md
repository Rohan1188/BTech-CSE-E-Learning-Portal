# 🎓 BTech CSE E-Learning Portal

Welcome to the **NotesLink** E-Learning Hub — a centralized, semester-wise portal for B.Tech Computer Science and Engineering (CSE) students to access curated YouTube video tutorials for every subject across all academic years.

Deployement : https://elearning.noteslink.in/

## 📌 Features

- 🎯 **Complete Curriculum Coverage**: Includes subjects from 1st to 8th semester.
- 🎥 **YouTube Video Tutorials**: Direct links to subject-wise playlists.
- 🧠 **Well-Organized Layout**: Clean, responsive design grouped semester-wise.
- 📱 **Mobile-Friendly**: Optimized for use on all screen sizes.
- 🔗 **Quick Access**: Jump straight to video content with one click.

## 🛠️ Tech Stack

- HTML5
- CSS3
- Font Awesome (for icons)
- Responsive Design (using Flexbox & Grid)

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/btech-cse-elearning.git
cd btech-cse-elearning
````

### 2. Open in Browser

You can directly open the `btech_cse_elearning.html` file in your browser:

```bash
start btech_cse_elearning.html  # Windows
open btech_cse_elearning.html   # macOS
xdg-open btech_cse_elearning.html  # Linux
```
